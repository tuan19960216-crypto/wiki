0找你的管理员要"专属链接"

在开始之前，你需要 2 条专属链接。它们看起来像这样：

vless://xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx@ip地址:443?encryption=none&flow=xtls-rprx-vision&security=reality&sni=...&type=tcp#🔒主路径
vless://yyyyyyyy-yyyy-yyyy-yyyy-yyyyyyyyyyyy@cdn.xxxx.ai:443?encryption=none&security=tls&type=ws&...#☁️备路径
🔑
怎么拿？ 在 Lark 私聊管理员（Gloom），写一句"给我发一下 VPN 链接"，他会私发给你。

为什么要私发？ 这两条链接相当于 VPN 的"密码"，泄露给陌生人会被白嫖流量，所以不能贴在公开页面。

拿到链接后，把它发到你自己的备忘录、Lark 收藏等地方备用。下面会让你复制粘贴。

📱 iPhone / iPad 配置

⚠️
必读：iOS 没有免费 VPN 客户端能用我们的协议。需要花 $2.99（约 ¥21）买一次「Shadowrocket」，终身使用。

重要前提：你的 Apple ID 必须是 美区 / 港区 / 台区（中国大陆区 App Store 没有这个 App）。如果你 ID 是大陆区，找管理员要个**借用账号**或自己注册一个美区 ID（注册免费）。
第 1 步

下载 Shadowrocket

切到美区 Apple ID → App Store → 搜索 Shadowrocket → 购买（$2.99）→ 等下载完成。

App Store 链接 →
第 2 步

导入第 1 条链接（主路径）

  1. 从你的 Lark / 备忘录里 复制管理员发给你的第 1 条 vless:// 链接（带 🔒主路径 字样的那条）
  2. 打开 Shadowrocket，主页右上角点"+"
  3. 它会自动弹窗问"是否添加剪贴板的服务器？" → 点 添加 / Add
  4. 主页就出现了一个 🔒主路径 节点（带美国国旗图标）
第 3 步

导入第 2 条链接（CDN 备路径）

同样的操作 —— 复制第 2 条 → 主页 + → 自动识别添加。

完成后主页应该看到 2 个节点：🔒主路径 和 ☁️备路径。

第 4 步

开启 VPN

  1. 主页**点选**"🔒主路径"（让它变成默认节点，前面有橙色小圆点）
  2. 主页顶部 开关打开（切到右边变蓝色）
  3. 系统会弹"是否允许添加 VPN 配置" → 允许（这个只首次会问）
  4. 开关变蓝 = 已连接
第 5 步（关键设置）

必开的隐私保护

主页左上角菜单图标 → 设置 / Settings：

  • 路由：配置 (Config) 模式（不要选 Global，不然连国内网站都走 VPN，慢）
  • DNS 模式：DNS Proxy，DNS 服务器填 1.1.1.1
  • Block QUIC 打开（防止某些应用绕过 VPN）
  • Bypass LAN 打开（局域网设备不走 VPN）

完成 → 跳到 第 3 步验证。

🤖 Android 配置

第 1 步

下载 V2rayNG（免费）

从 GitHub 直接下 APK 安装：

下载最新版 V2rayNG →

找到 v2rayNG_x_x_x_arm64-v8a.apk（绝大多数手机都是 arm64）下载后安装。

第 2 步

导入第 1 条链接

  1. 复制管理员给你的第 1 条 vless:// 链接
  2. 打开 V2rayNG → 右上角 + → 从剪贴板导入批量 URL
  3. 看到主页出现"🔒主路径"节点
  4. 同样方式导入第 2 条"☁️备路径"
第 3 步

开启 VPN

  1. 选中 🔒主路径（点一下，节点变绿）
  2. 主页右下角 大圆形按钮 V 点击
  3. 系统弹窗"是否允许 VPN 连接" → 确定
  4. 按钮变成红色 V = 已连接

完成 → 跳到 第 3 步验证。

💻 Windows 配置

第 1 步

下载 Clash Verge Rev（免费）

下载最新版 →

选 Clash.Verge_x_x_x_x64-setup.exe（64位 Windows） → 双击安装 → 一路下一步。

第 2 步

导入链接

  1. 复制管理员给你的第 1 条 vless:// 链接
  2. 打开 Clash Verge → 左侧 "代理 / Proxies"
  3. 右上角 "导入 / Import" → 选 从剪贴板导入
  4. 同样导入第 2 条
  5. 点选 🔒主路径
第 3 步

开启系统代理

  1. 左侧 "设置 / Settings"
  2. 打开 "系统代理 / System Proxy" 开关
  3. 打开 "TUN 模式 / TUN Mode" 开关（可让浏览器以外的应用也走 VPN）

完成 → 跳到 第 3 步验证。

🍎 Mac 配置

第 1 步

下载 ClashX Meta（免费）

下载最新版 →

选 ClashX.Meta.dmg（M 系列 Mac 选 arm64，Intel Mac 选 x64）→ 拖到"应用程序"。

第 2 步

导入链接

  1. 复制管理员给你的第 1 条 vless:// 链接
  2. 打开 ClashX Meta（菜单栏会出现一个小猫图标）
  3. 菜单栏小猫 → 配置 / Configurations → 从 URL 导入
  4. 同样导入第 2 条
第 3 步

开启系统代理

  1. 菜单栏小猫 → 设置为系统代理 / Set As System Proxy（打勾）
  2. 菜单栏小猫 → 出站规则 / Outbound Mode → 选 规则 / Rule

完成 → 跳到 第 3 步验证。

3验证 VPN 是否生效

VPN 连接好后，做下面 1 个测试 确认成功：

访问下面这个网址

https://ipinfo.io/ip

用已开 VPN 的设备的浏览器（Safari / Chrome 都行）打开。

  • ✅ 看到一个 104. 开头 或者 美国 IP = VPN 成功
  • ❌ 看到 中国 IP（如 116. / 121. / 223. 开头） = VPN 没生效
🎉
显示美国 IP 后，立刻测试 https://www.google.com 能不能打开。能打开 → 你已经能用全球互联网了！

常见问题

Q: 连上了 VPN，但还是打不开 Google？

检查：

  1. 路由模式是否选错了 —— 不要选 Global / 全局，要选 Rule / 规则 / Config 模式
  2. 关闭浏览器再重开（浏览器有 DNS 缓存）
  3. 切换到 ☁️备路径节点试试，如果备路能用 = 主路径被运营商干扰，正常
Q: 速度很慢？

正常情况：

  • 主路径延迟 150-250ms（白天/上班时间）
  • 晚上 8-11 点 国内出口拥塞，可能 300ms+，切到 ☁️备路径会更稳
  • 如果常年慢，告诉管理员"我这里 VPN 速度很慢"
Q: 突然连不上了？

步骤：

  1. 切换到 ☁️备路径节点试试 —— 如果它能用 = 主 IP 被封了，立刻告诉管理员
  2. 都用不了 —— 重启路由器 / 切到手机 4G 流量再试一次（排除你本地网络问题）
  3. 都不行 —— 在 Lark 群里 @管理员 "VPN 全部连不上"
Q: 流量限制？

整个团队共享 2 TB / 月 流量，正常工作 + 看视频完全够用。

请不要用 VPN 下载大型游戏 / 4K 高清电影 / 持续挂 YouTube 等高流量行为。

Q: 公司之外的设备能用吗？比如家里的电脑？

能用，没有"绑定公司网络"这种限制。但请仅限本人使用，不要分享给家人或朋友 —— 多一个人用就多一份流量消耗。

Q: 手机切换地点（比如 4G ↔ WiFi），VPN 会断吗？

会自动重连。如果没重连，手动关一下 VPN 开关再开就行。

Q: 我可以同时用多台设备吗？

可以。一份链接可以同时在你的手机+电脑+iPad 上用，不冲突。

Q: 离职后还能用吗？

不能。离职时管理员会吊销你的"专属链接"，从那时起这两条 vless:// 都会失效。
