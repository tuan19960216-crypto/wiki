Claude Code 是什么？

Claude Code 是 Anthropic 官方出的命令行工具，让你能在终端里直接跟 Claude（一个顶级 AI 模型）协作写代码、改 bug、跑脚本。它不是网页版，是装在你电脑上跑的 CLI。

为什么在国内用不了？

Anthropic 不向中国大陆、香港、俄罗斯等地区开放服务，所以：

  • 直连 api.anthropic.com 会返回 403 Forbidden
  • 注册账号也会因为 IP 被拒
  • 必须全程挂梯子（VPN），而且不是随便一个就行，后面会讲

你需要准备

  • 一台 Windows 10 或 11 电脑
  • 团队 VPN 节点的"专属链接"（在 Lark 私聊管理员获取，配置流程见 VPN 一键上手指南）
  • 一个海外手机号（用于注册，可用 Google Voice 或临时号）
  • 一张海外信用卡或虚拟卡（如果要付费订阅 Pro/Max）
  • 40–60 分钟耐心
订阅还是按量付费？

如果你每天用得多，强烈建议直接订阅 Claude Pro（$20/月） 或 Max（$100/月），比按 API token 计费划算 5–10 倍。这份教程按"订阅 Pro"流程来讲。

1搞定 VPN 网络

最重要的一步

90% 的新手卡在这里。后面的每一步都依赖这一步配好。团队已经搭好了一个稳定的自建 VPN 节点（美国 LA 独立 IP，Anthropic 不会风控），下面是 Windows 端的最简配置流程。

1.1 用团队自建 VPN 节点（已就绪）

关键信息

团队 VPN 节点：美国 LA 独立 IP，VLESS-Reality + Cloudflare 双路径，2TB / 月共享流量。
具体的 vless:// 链接不公开（防陌生人薅流量）—— 在 Lark 私聊管理员索取你的"专属链接"。

👉 各设备完整安装/导入步骤已单独成文，跟着做就行：

📖 打开 VPN 一键上手指南 →

下面只列出 Windows 端 + Claude Code 特有的额外要求（TUN 模式、命令行环境变量），通用配置请看上面那篇。

1.2 Windows 客户端 = Clash Verge Rev

  1. 下载 Clash Verge Rev 最新版（Clash.Verge_x.x.x_x64-setup.exe）→ 双击安装
  2. 按"VPN 一键上手指南"的 Windows 章节导入两条 vless:// 链接
  3. 选中 🔒Reality-Main 节点
  4. 设置页打开 系统代理 (System Proxy) 开关
  5. 必开 TUN 模式（设置 → 开启 TUN / 虚拟网卡）—— 让 PowerShell / Git / Claude Code 也走代理
关键设置（常被忽略）

不开 TUN 模式，Claude Code 在 PowerShell 里跑时不会走你的 VPN。表面上浏览器能访问 Google，但 claude 命令会一直 timeout，怀疑人生。

1.3 给 PowerShell 显式配代理（保险措施）

开了 TUN 通常已经够了，但加上环境变量更稳。按 Win+R，输入 sysdm.cpl，回车，然后：

  1. 切换到"高级"标签 → 点"环境变量"
  2. 在"用户变量"里点"新建"，添加两条：
    • 变量名 HTTPS_PROXY，值 http://127.0.0.1:7897
    • 变量名 HTTP_PROXY，值 http://127.0.0.1:7897
  3. Clash Verge Rev 默认混合代理端口是 7897（设置页能看到，自定义改了的话填你改的那个）
  4. 重启所有终端窗口让变量生效

1.4 验证网络真的通了

打开 PowerShell，跑：

curl -v https://api.anthropic.com
  • ✅ 返回 401 Unauthorized 或一段 JSON —— 通了（401 是因为没带 API Key，正常）
  • ❌ timeout / connection refused —— 代理没生效，回头检查 TUN 和环境变量
  • ❌ 403 Forbidden: unsupported region —— 节点不在美国（不应该出现，团队节点就在美国 LA；如果真出现联系管理员）

多做一步外网验证：浏览器打开 https://ipinfo.io/ip，应该看到一个 104. 开头的美国 IP。看到中国 IP 说明代理没生效。

2注册 Anthropic 账号

全程挂美国节点

注册过程不要切换节点、不要断开 VPN，否则会触发风控。用浏览器隐身模式注册。

2.1 注册步骤

  1. 打开浏览器隐身窗口，访问 https://claude.ai
  2. 用 Gmail 或 Outlook 邮箱注册（不要用 QQ、163 等国内邮箱）
  3. 收到验证邮件 → 点链接验证
  4. 需要手机号验证：
    • 有海外实卡最好
    • 没有就用 Google Voice（免费，但需要翻墙 + 一次性美国号）
    • 或用 SMS-Activate 等一次性接码平台（几块钱）

2.2 订阅 Pro（可选但推荐）

  1. 登录后点左下角头像 → "Upgrade Plan" → 选 Pro ($20/月) 或 Max ($100/月)
  2. 填信用卡信息：
    • 有海外信用卡最好
    • 没有可用虚拟卡：WildCard、NobePay、OneKey Card（国内很多人在用）
    • 地址填一个美国地址（虚拟卡平台会给你一个）
  3. 付款成功后，Claude Code 就能用订阅额度登录，不需要额外买 API 额度

3安装 Git for Windows

为什么要装 Git？

Claude Code 在 Windows 上运行时，底层会调用 Git 自带的 Bash 来执行命令（比如读写文件、跑脚本）。没有 Git for Windows，Claude Code 启动就会报错。

安装步骤

  1. 访问 https://git-scm.com/downloads/win
  2. 下载"64-bit Git for Windows Setup"
  3. 双击运行，全程点"Next"默认选项就行，不用改任何设置
  4. 装完后，按 Win+R 输入 cmd 回车，输入： 
    git --version
    看到类似 git version 2.43.0.windows.1 就说明装好了。

给 Git 配代理（重要）

打开 PowerShell 或 Git Bash，跑：

git config --global http.proxy http://127.0.0.1:7890
git config --global https.proxy http://127.0.0.1:7890

以后 git clone 才不会超时。

4安装 Claude Code

方法 A：官方一键安装脚本（推荐）

  1. 按 Win 键，搜 PowerShell，右键点"以管理员身份运行"
  2. 粘贴并回车： 
    irm https://claude.ai/install.ps1 | iex
  3. 等待下载（需要 VPN 正常工作），大约 1–3 分钟
  4. 安装结束后会提示你重启终端
如果报错"脚本无法运行"

先跑这条命令允许 PowerShell 执行脚本：

Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

输入 Y 确认，然后再跑上面的安装命令。

方法 B：npm 安装（备选，需要先装 Node.js ≥ v18）

npm install -g @anthropic-ai/claude-code

注意：不要加 sudo（Windows 本来也没有 sudo）。

验证安装

重启 PowerShell，跑：

claude --version

看到版本号就算成功。如果提示"不是内部命令"，重启电脑试试（PATH 环境变量没刷新）。

5首次登录 & 试运行

5.1 创建一个测试项目

打开 PowerShell，新建一个文件夹并进去：

mkdir C:\dev\claude-test
cd C:\dev\claude-test
路径别用中文

不要把项目放在 C:\我的文档\测试 这种带中文的路径下，会乱码报错。用 C:\dev\xxx 这种纯英文路径。

5.2 启动 Claude Code

claude

第一次启动会让你选登录方式：

  • Claude Pro/Max subscription —— 如果你订阅了 Pro/Max，选这个（免费额度额大）
  • Anthropic Console (API key) —— 按 token 计费，适合重度用户

选完之后会自动打开浏览器让你登录。登录成功后浏览器会显示"You can close this window"。

登录失败了？

浏览器回调要跳回 localhost。如果你用了带代理插件的浏览器（比如 SwitchyOmega），先关掉所有代理插件，用系统默认的 Edge 或 Chrome 打开。

5.3 试用

登录成功后，你会看到一个对话界面。试试输入：

帮我写一个 Python 脚本，打印 Hello World

Claude 会直接生成文件。如果能跑到这里，恭喜你，全部配置完成 🎉

常见报错速查表

报错原因解决
403 Forbidden / unsupported region 节点是香港/国内，或 VPN 没生效 换美/日节点，确认 TUN 开启，重启终端
ECONNRESET / timeout 代理不稳定，或端口不对 换节点；检查 HTTPS_PROXY 端口和 Clash 实际端口一致
claude 不是内部命令 PATH 没刷新 重启终端；还不行就重启电脑
Git Bash not found 没装 Git for Windows 回到第 3 步装 Git
登录回调失败 浏览器代理插件冲突 用系统默认浏览器，关闭所有代理插件
中文乱码 CMD 默认 GBK 编码 换 Windows Terminal（微软商店免费下），或跑 chcp 65001
安装脚本"被禁用" PowerShell 执行策略太严 Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
账号突然被封 共享 IP 节点被风控 换独立 IP 节点；已封的账号基本救不回来，只能重新注册

万能诊断命令

任何报错，先跑这个自检：

claude doctor

它会一次性检查网络、Git、Node、PATH 等所有项，告诉你具体哪里出问题。

进阶技巧 & 省钱指南

终端选择

  • 用 Windows Terminal（微软商店免费）—— UTF-8 支持好、多标签、颜色漂亮
  • 或直接用 Git Bash —— Claude Code 内部就是用 Bash 跑命令，行为最一致
  • 老的 CMD —— 编码和显示都有问题
  • PowerShell ISE —— 已被微软淘汰

VPN 常驻挂着

Claude Code 是流式输出（一边生成一边显示），VPN 断几秒就会中断当前对话。所以：

  • Clash Verge Rev 设置"开机自启"（设置 → 启动设置 → 开机自启动）
  • 始终连 🔒Reality-Main 主节点（团队节点延迟最低）；只在它失效时手动切到 ☁️CDN-Backup
  • VPN 状态有问题时，在 Lark 群 @管理员 —— 监控会同步告警，通常我们已经在处理

省钱技巧

  • Pro 订阅比 API 按量付费便宜 5–10 倍，每天用就订阅 Pro
  • 订阅 Max ($100/月) 送 5x 使用额度，团队用或重度用户直接上 Max
  • 不要乱跑大任务，Claude Code 每次发送会带上下文，上下文越长越贵（API 模式下）

安全建议

  • API Key 存在 ~/.claude/ 下，不要把这个目录同步到 OneDrive/GitHub
  • 别在公司电脑上装公司不允许的 VPN，违反合规会被开除
  • 别用同一个账号频繁换 IP 登录，会触发风控

想更省心？用 WSL 2

如果你本来就写 Linux 代码，或者被 Windows 路径折磨够了，直接上 WSL 2（Windows Subsystem for Linux）：

wsl --install

装完 Ubuntu 后进入 WSL 终端，跑：

curl -fsSL https://claude.ai/install.sh | bash

注意 WSL 不会自动继承 Windows 的代理，要在 ~/.bashrc 里加：

export HTTPS_PROXY=http://$(ip route | grep default | awk '{print $3}'):7897
export HTTP_PROXY=$HTTPS_PROXY

端口 7897 是 Clash Verge Rev 默认（老版 Clash for Windows 是 7890）。同时要在 Clash 设置里勾选"允许局域网连接 / Allow LAN"。
