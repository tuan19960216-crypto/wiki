ACT 03 · 第一句话2026-05-18　21:33 CST · 触达延迟 3h27m

四段式破冰 · 软定价 + 二选一

首条 AI DM 在源料 pack 内（#1）。这条破冰的特点是软定价（"按投影算千把一平"）+ 二选一反问（"板材还是五金"）。客户问价但 AI 没硬报数字 — 用投影计价方式给一个心理锚（千把一平），同时把决策权切到客户的偏好维度。

① 招呼 + 视频呼应
复述视频痛点
「油工流程里墙面找平防开裂是关键」
② 省级接业务
省级归属感
「陕西最近问全屋柜体的多」
③ 软定价 + 卖点
投影计价 + 直供 + 紧迫
「按投影算千把一平，一线品牌直供」
④ 二选一反问
板材 vs 五金
「你最在意板材还是五金？」
软定价话术模式 · ice_break 询价标配
客户直接问价位 — AI 不能拒答（会显得藏价不真诚），也不能硬报具体单价（容易被横向比价 + 失去后续谈话筹码）。软定价：「按投影算千把一平」给的是计价方式 + 数量级，不是具体价 — 客户读到知道这家有透明计价习惯，同时无法直接比价。结尾二选一反问把客户从"先看价"切到"先看自己的偏好"，让对话从价格谈判转回需求挖掘。

5 个标准 product 询价的话术原则

1
「什么价位」是 product type 的标准入场券。老谢这一句直接把 LLM 标签锁定到 product · medium · active 三轴 78 分 — 评分一过 70，conversation 被自动建出来、进 ice_break 流水线。这是从评论到对话的关键 gate，跟用户3824342698628 / 空间向量 / 即将是个辣妹同分。
2
询价类破冰必须用软定价 · 不能拒答也不能硬报数字。客户直接问价，AI 用「按投影算千把一平」给计价方式 + 数量级，没给具体单价。计价方式 = 显专业，数量级 = 给心理锚，但都不可直接横向比价。软定价 = 用变量分解代替硬数字，让客户感觉透明又留住后续谈话筹码。
3
IP 属地是免费的地域钩子。「陕西最近问全屋柜体的多」— 数据本就在评论 API 里，不用反问、不用客户额外披露。在客户没自报市的情况下，省级回退是 ice_break 标配 — 知识号 + 全国服务市场景下省级是唯一可用 anchor。
4
触达延迟 3h27m · 端到端流水线时延的健康基线。评论 18:06 留下 → DM 21:33 推过去 — 抓评论 → LLM 评分 → 起草 → 真机推送整条线路 3h27m。这是 ice_break 阶段的可接受基线（跨夜 17h 已经差、跨天 / 跨周更差）。监控指标可以围绕这个数字定 P50/P90。
5
stage 真实卡在哪显示哪 · 不要为"完整结局"假装。ice_break 没继续推进就显示 ice_break，sec_uid + sent_at + comment_id 都可追溯 — 案例库不为"完整结局"而隐藏当下真实状态。下一次轮询拉到客户回复，本页 ACT 05 / 06 自然展开。诚实展示当下状态本就是案例库的价值。
