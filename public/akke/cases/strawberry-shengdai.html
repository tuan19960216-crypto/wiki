复盘要点 · 这条线索教会我们的

  • 评论里就有产品型号 → 直接复述。「508 这型号近期问的人多」让客户一眼觉得你听懂了，比泛聊「全屋定制」更落地。
  • IP 属地是免费的破冰利器。「湖北我接过几单类似选型」一句话拉近距离 — 数据本来就在评论 API 里，不用反问。
  • 反问要给条件而不是「价格多少」。「你家厨房预留尺寸多大，橱柜定了没？」让客户主动 disclose 户型 — 她真的回了「我没有做嵌入 / 放客厅里」，把户型信息送到我们手上。
  • 第二轮要承接客户给的条件。客户说「放客厅」我们立刻接「做整面包柜把空间收齐」— 没有把对话拉回我们想说的，而是顺着客户的现实推方案。
  • 加微锚点放第二轮，不放第一轮。先让客户 disclose，再用「留个v发实拍型号 + 活动价单」把信任额度兑现成微信号，不会被当成营销号一眼拒。
